Step Patch

把你輸入的步數，寫進 iPhone「健康」App。

在 GitHub 查看專案 安裝說明

它能做什麼

用最簡單的流程補上步數，並在 App 內查看與管理紀錄。

  • 快速選擇或手動輸入要新增的步數
  • 指定紀錄時間後寫入「健康」
  • 查看每日統計與當日明細
  • 刪除由 Step Patch 寫入的紀錄
  • 支援繁體中文與英文

怎麼開始

用 Xcode 開啟專案，簽章後裝到實體 iPhone 即可。

Clone 專案並用 Xcode 開啟 StepEditor.xcodeproj

選擇你的 Team，確認已啟用 HealthKit

連上 iPhone 後 Run，允許步數寫入權限
