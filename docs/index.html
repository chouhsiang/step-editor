Step Patch

把你輸入的步數，寫進 iPhone「健康」App。

查看 GitHub 專案 了解功能

專為補步數而生

從輸入、同步到查閱紀錄，流程清楚，介面乾淨。

01

快速新增

用快捷按鈕或手動輸入步數，一鍵寫入「健康」App。

02

指定時間

可選擇紀錄日期與時間，讓步數對應到正確時段。

03

每日統計

從健康資料讀取近況，用列表快速掌握每天總步數。

04

明細與刪除

點進單日查看每筆紀錄，並可刪除本 App 寫入的資料。

05

中英雙語

依系統語言自動切換繁體中文與英文介面。

06

開源授權

採 MIT License，歡迎 fork、修改與分享給更多人。

開始使用

用 Xcode 開啟專案，裝到實體 iPhone 就能同步到「健康」。

開啟專案

Clone 後用 Xcode 開啟 StepEditor.xcodeproj。

設定簽章

選擇你的 Team，並確認已啟用 HealthKit。

安裝到手機

連上 iPhone 後按 Run，允許步數寫入權限即可。

在 GitHub 開源

原始碼、說明與授權都公開，歡迎一起改進 Step Patch。
